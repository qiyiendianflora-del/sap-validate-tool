# SAP凭证校验工具

SAP财务凭证B表必输项校验工具，支持多规则校验，PyQt5 GUI界面。

## 功能
- ✅ 必输项校验（科目→必输字段）
- ⚖️ 借贷平衡校验
- 📋 统驭科目 vs 记账码校验
- 🚫 非6开头科目禁填字段校验
- 📊 费用类别组合校验

## 下载
进入 [Releases](../../releases) 页面，下载最新的 `SAP凭证校验工具.exe`，双击运行。

## 使用
1. 上传B表（待校验凭证Excel）
2. 上传科目必输校验规则表
3. 上传费用mapping规则表（可选，首次上传后自动记住）
4. 点击「开始校验」
5. 查看结果，点击「下载校验报告」导出标红Excel
